Etapa 1: Assinatura Claude

O Claude é a inteligência artificial que faz o Mega Brain funcionar. Sem uma assinatura ativa, o sistema não tem como processar seus materiais de conhecimento.

Atenção
O plano Free do Claude NÃO funciona com o Claude Code. Você precisa do plano Pro ou Max para usar o Mega Brain.
Pro
Plano Pago

Funciona para uso normal do Mega Brain. Tem limites mensais de uso -- bom para começar e testar o sistema. Consulte os valores em claude.com/pricing.

Max
Plano Avançado

Recomendado para uso intenso. Sem limites de uso mensal -- ideal se você usa o pipeline todo dia. Consulte os valores em claude.com/pricing.

Qual plano escolher?
Se você está começando agora, o plano Pro é suficiente. Você pode trocar para o Max depois, se sentir necessidade.

Acesse o site do Claude para criar sua conta e escolher seu plano:

Acessar claude.ai

Passo a passo

  1. Acesse claude.ai
  2. Clique em "Start for free"
  3. Crie sua conta com email ou Google
  4. Escolha o plano Pro ou Max
  5. Confirme o pagamento

Como verificar

Acesse claude.ai e você deve ver seu painel com conversas. Se ver a tela de login, sua conta ainda não está ativa.

Não recebi o email de confirmação

  1. Verifique a pasta de spam ou lixo eletrônico
  2. Espere até 5 minutos -- alguns provedores demoram
  3. Se não chegar, tente criar a conta com outro email (Gmail costuma funcionar melhor)
  4. Tente o login com Google diretamente, que não precisa de confirmação por email

Erro no pagamento

  1. A Anthropic aceita cartões internacionais. Confira se seu cartão tem compras internacionais habilitadas
  2. Se o cartão for recusado, tente um cartão virtual (Nubank, Wise ou Inter funcionam)
  3. Verifique se o endereço de cobrança está preenchido corretamente
  4. Alguns bancos bloqueiam a primeira tentativa por segurança -- tente novamente após aprovar a notificação do banco
~8 minutos

Etapa 2: VS Code + Claude Code

Na etapa anterior, você criou sua conta Claude -- a inteligência artificial que vai fazer o Mega Brain pensar.
Agora vamos instalar o VS Code, que é o escritório onde você vai trabalhar. Tudo o que você fizer no Mega Brain vai acontecer dentro do VS Code.
Importante
NÃO instale o "Visual Studio" (sem "Code"). São programas completamente diferentes. O correto é Visual Studio Code.
O que é o VS Code?
O VS Code é um editor de código gratuito criado pela Microsoft. Ele é onde você vai abrir, editar e rodar o Mega Brain. Pense nele como a bancada de trabalho onde tudo acontece.

1. Baixar e instalar o VS Code

Acesse a página de download do VS Code:

https://code.visualstudio.com/download
  1. Clique em "Download for Windows"
  2. Execute o instalador que foi baixado
  3. Clique Next em todas as telas. Você não precisa mudar nenhuma opção
  4. Na última tela, clique em Finish

Acesse a página de download do VS Code:

https://code.visualstudio.com/download
  1. Clique em "Mac Universal"
  2. Abra o arquivo .zip que foi baixado
  3. Arraste o Visual Studio Code para a pasta Applications

2. Instalar a extensão Claude Code

A extensão Claude Code conecta o VS Code com a inteligência artificial que você contratou na etapa anterior. É por meio dela que o Mega Brain funciona.

  1. Abra o VS Code
  2. Clique no ícone de Extensions na barra lateral esquerda (ou pressione Ctrl+Shift+X)
  3. Na caixa de busca, digite "Claude Code"
  4. Clique no resultado "Claude Code" publicado pela Anthropic
  5. Clique em "Install"

3. Fazer login na extensão

Após instalar, o VS Code vai pedir pra você fazer login com sua conta Claude. Clique em "Sign in with Claude" e autorize o acesso.

Por que o login?
O login conecta a extensão com sua assinatura Claude. Sem ele, o VS Code não consegue acessar a inteligência artificial.

Como verificar

Abra o terminal do VS Code (Ctrl+`) e digite:

claude

Claude Code é mostrado no terminal e pede seu primeiro prompt.

Não encontrei a extensão Claude Code

  1. Verifique se você está na versão mais recente do VS Code (menu Help > Check for Updates)
  2. Confira se digitou exatamente "Claude Code" na busca (sem espaço extra)
  3. Procure pela extensão publicada pela Anthropic -- ignore resultados de outros publicadores

A extensão instalou mas não aparece no VS Code

  1. Feche o VS Code completamente
  2. Abra novamente
  3. Verifique se a extensão aparece na lista de extensões instaladas (ícone de Extensions na barra lateral)

O VS Code não abriu o terminal com Ctrl+`

  1. O atalho pode variar dependendo do teclado. Tente o menu View > Terminal
  2. No Mac, o atalho é Cmd+` (Command + crase)
  3. Se nenhum atalho funcionar, use o menu Terminal > New Terminal
~5 minutos

Instalar Node.js

Na etapa anterior, você instalou o VS Code com a extensão Claude Code -- seu escritório de trabalho está pronto.
Agora vamos instalar o Node.js. É o motor que faz o Mega Brain funcionar por baixo dos panos -- sem ele, o sistema não roda.

O Node.js é o motor que executa o Mega Brain. Pense nele como o motor de um carro -- você não vê, mas sem ele nada anda.

O que é LTS?
Vamos instalar a versão LTS (Long Term Support). É a versão mais estável e recomendada -- a que recebe atualizações por mais tempo. Se você já tem Node.js 18+ instalado, pule direto para a verificação no final desta etapa.

Acesse o site oficial do Node.js:

https://nodejs.org
  1. Clique no botão verde LTS
  2. Execute o instalador e clique Next-Next-Finish em tudo. Não mude nenhuma opção.
  3. Feche e reabra o VS Code depois de instalar

Acesse o site oficial do Node.js:

https://nodejs.org
  1. Clique em LTS para Mac
  2. Abra o arquivo .pkg baixado e siga o instalador
  3. Feche e reabra o VS Code depois de instalar

Como verificar

Abra o terminal do VS Code (Ctrl+`) e digite:

node --version

Deve mostrar uma versão 18 ou maior, como v20.11.0

Se deu certo

$ node --version v20.11.0

Se aparecer isso

$ node --version node : O termo 'node' não é reconhecido...

'node' não reconhecido

Feche e reabra o VS Code completamente. Se ainda assim não funcionar, reinicie o computador e tente novamente.

Versão antiga do Node (menor que 18)

Desinstale a versão atual pelo Painel de Controle (Windows) ou via Terminal (Mac), depois instale a versão LTS mais recente.

~5 minutos

Instalar Python

Na etapa anterior, você instalou o Node.js -- o motor que faz o Mega Brain funcionar. Percebeu que foi rápido?
Agora vamos instalar o Python, que cuida dos scripts de inteligência artificial. Mesma lógica: baixar, instalar, verificar. Tem UM detalhe importante nesta etapa -- veja o alerta vermelho logo abaixo. Fora isso, é idêntico.

O Python executa os scripts de inteligência artificial do Mega Brain. É ele que faz a mágica acontecer nos bastidores.

PARE AQUI ANTES DE CLICAR EM INSTALAR

Na primeira tela do instalador, lá embaixo, tem um checkbox escrito "Add python.exe to PATH". Esse checkbox vem DESMARCADO.

Você precisa MARCAR ele antes de clicar em "Install Now".

Se instalar sem marcar, o Python fica instalado mas o seu computador não sabe onde encontrar ele. Aí nada funciona.

(Se já instalou sem marcar: desinstale, e instale de novo marcando o checkbox desta vez. Não tem problema.)

Acesse a página de downloads do Python:

https://www.python.org/downloads/
  1. Clique no botão amarelo Download Python 3.12.x
  2. Abra o arquivo baixado
  3. MARQUE o checkbox 'Add python.exe to PATH' (veja o alerta acima)
  4. Clique em Install Now e aguarde

No Mac, Python 3 pode já vir instalado. Vamos verificar isso primeiro.

python3 --version
Resultado
Se apareceu uma versão 3.10 ou maior (exemplo: Python 3.12.1), você já tem Python instalado! Pode pular para a verificação abaixo.

Se não apareceu nada ou apareceu erro, baixe o Python:

https://www.python.org/downloads/
  1. Clique em Download Python 3.12.x para Mac
  2. Abra o arquivo .pkg baixado e siga o instalador

Como verificar

Abra o terminal do VS Code (Ctrl+`) e digite:

python --version

No Mac, tente python3 --version se python não funcionar

Deve mostrar Python 3.10.x ou maior

Se deu certo

$ python --version Python 3.12.1

Se aparecer isso

$ python --version 'python' não é reconhecido como um comando interno

Se apareceu a versão (3.10 ou maior), está perfeito! Se apareceu erro, veja 'Problemas Comuns' abaixo.

'python' não é reconhecido

Isso significa que o PATH não foi configurado. Desinstale o Python, instale novamente e desta vez MARQUE o checkbox 'Add to PATH' na primeira tela.

Aparece versão 2.x

Você tem uma versão antiga. Tente digitar python3 --version. Se funcionar, use python3 em vez de python nos próximos passos.

Instalação pede permissão de administrador

Clique com botão direito no instalador e selecione 'Executar como administrador'.

~5 minutos

Etapa 5: Baixar o Mega Brain

Nas etapas anteriores, você instalou todas as ferramentas base: Claude, VS Code, Node.js e Python.
Agora vamos trazer o projeto Mega Brain para o seu computador. É um comando só no terminal.
Antes de começar esta etapa:
  • Etapa 1: Assinatura Claude
  • Etapa 2: VS Code + Claude Code
  • Etapa 3: Node.js instalado
  • Etapa 4: Python instalado
PRE-REQUISITO: Git (so para Windows)

O Windows não vem com Git instalado. Antes de continuar, vamos verificar se você já tem.

Abra o terminal do VS Code (Ctrl+`) e digite:

git --version

Se mostrou uma versão

$ git --version git version 2.40.0.windows.1

Se aparecer isso

$ git --version git : O termo 'git' não é reconhecido...

Se mostrou uma versão (ex: git version 2.40.0): você já tem Git. Pode pular este bloco.

Se apareceu erro ('git' não reconhecido): siga os passos abaixo para instalar.

  1. Acesse git-scm.com/downloads
  2. Baixe o instalador para Windows
  3. Execute o instalador com Next-Next-Finish em tudo. Não mude nenhuma opção.
  4. Feche e reabra o VS Code
  5. Tente git --version novamente -- agora deve funcionar

Você não precisa de conta no GitHub para baixar o Mega Brain. O repositorio é público e qualquer pessoa pode fazer o download.

Abra o terminal do VS Code (Ctrl+`) e cole este comando:

O comando git clone copia o projeto inteiro para o seu computador:

git clone https://github.com/thiagofinch/mega-brain.git

Aguarde o download completar -- você verá mensagens de progresso no terminal. Quando terminar, o terminal vai voltar para o prompt normal.

Abrir a pasta no VS Code

Agora abra a pasta no VS Code:

  1. Clique em File no menu superior
  2. Selecione Open Folder
  3. Navegue até encontrar a pasta mega-brain
  4. Clique em Open (ou Selecionar Pasta no Windows)

Como verificar

O VS Code vai mostrar os arquivos do Mega Brain no Explorer (painel esquerdo). Você deve ver pastas como src, docs, e arquivos como package.json.

Se deu certo

$ git clone https://github.com/thiagofinch/mega-brain.git Cloning into 'mega-brain'... remote: Enumerating objects: 150, done. remote: Total 150, done. Receiving objects: 100% (150/150), done.

Se aparecer isso

$ git clone https://github.com/thiagofinch/mega-brain.git fatal: repository not found

'git' não reconhecido

Siga os passos do bloco PRE-REQUISITO: Git acima para instalar o Git.

Demora muito / travou

Verifique sua conexão com internet. Tente cancelar (Ctrl+C) e rodar o comando novamente.

Permission denied

Tente fechar e reabrir o VS Code como administrador.

~8 minutos

Etapa 6: Obter suas Chaves de Acesso (API Keys)

Na etapa anterior, você baixou o projeto Mega Brain para o seu computador.
Agora vamos obter as chaves de acesso aos serviços de IA. São duas chaves no total -- cada uma leva cerca de 4 minutos para criar.
Antes de começar esta etapa:
  • Etapa 5: Mega Brain baixado

Uma API Key é como a senha do Wi-Fi da sua casa. Sem ela, o Mega Brain não conecta na inteligência artificial. Você vai precisar de duas chaves: uma da Anthropic (para o Claude, a IA principal) e uma da OpenAI (para o Whisper, que transcreve áudios).

Sobre custos
Cada plataforma cobra por uso da API. Adicione um valor inicial de crédito em cada uma — um valor pequeno já é suficiente para semanas de uso normal do Mega Brain. Consulte os preços diretamente em cada plataforma.
Segurança das chaves
Guarde suas chaves em um lugar seguro -- um bloco de notas ou gerenciador de senhas. Você vai precisar delas na próxima etapa. Nunca compartilhe suas chaves com ninguém.

Chave 1: Anthropic (Claude)

  1. Acesse console.anthropic.com
  2. Crie uma conta ou faça login
  3. No menu lateral, clique em API Keys
  4. Clique em Create Key
  5. Digite um nome para a chave (ex: 'mega-brain')
  6. Copie a chave gerada e guarde em um lugar seguro -- ela NÃO vai aparecer novamente

Chave 2: OpenAI (Whisper)

  1. Acesse platform.openai.com
  2. Crie uma conta ou faça login
  3. Clique em API Keys no menu
  4. Clique em Create new secret key
  5. Adicione um nome (ex: 'mega-brain')
  6. Copie a chave e guarde -- ela também NÃO vai aparecer novamente
  7. Adicione crédito em Billing > Add payment method (consulte os valores na plataforma)

Como verificar

Você deve ter duas chaves copiadas e guardadas:

  • ANTHROPIC_API_KEY (começa com 'sk-ant-...')
  • OPENAI_API_KEY (começa com 'sk-...')

Você vai usar essas duas chaves na próxima etapa.

Não tenho cartão de crédito internacional

A Anthropic aceita cartões nacionais (Visa/Mastercard). A OpenAI pode exigir cartão internacional -- neste caso, um cartão pré-pago internacional resolve.

A chave já sumiu antes de copiar

Você precisará criar uma nova chave. Clique em 'Create Key' novamente e desta vez copie imediatamente.

Não sei onde colocar a chave agora

Você NÃO precisa usar as chaves agora. Guarde em um bloco de notas. Vamos usá-las na próxima etapa.

~5 minutos

Etapa 7: Configurar o Mega Brain

Na etapa anterior, você obteve suas duas API Keys -- as chaves de acesso aos serviços de IA.
Agora vamos juntar tudo. Primeiro instalamos as dependências do projeto, depois rodamos o assistente de configuração que vai usar suas chaves.
Antes de começar esta etapa:
  • Etapa 5: Mega Brain baixado
  • Etapa 6: API Keys obtidas

Passo 1: Instalar as Dependências

No terminal do VS Code, certifique-se de que você está dentro da pasta mega-brain. O terminal deve mostrar algo como /mega-brain$ no final da linha.

Verifique a pasta
Certifique-se de que o terminal está dentro da pasta mega-brain antes de rodar este comando. Se você estiver em outra pasta, o comando vai dar erro.

Digite o comando abaixo e pressione Enter:

npm install

Este comando baixa todas as peças que o Mega Brain precisa para funcionar. Pode levar de 1 a 3 minutos. Você vai ver muitas mensagens passando -- isso é normal.

Se deu certo

$ npm install added 247 packages, audited 248 packages in 45s found 0 vulnerabilities

Se aparecer isso

$ npm install npm: command not found -- ou -- ENOENT: no such file or directory, open 'package.json'

Passo 2: Rodar o Assistente de Configuração

Agora rode o assistente de configuração:

npx mega-brain-ai setup

O assistente vai fazer perguntas sobre suas API Keys. Para cada pergunta, cole a chave correspondente e pressione Enter.

Nada aparece quando colo?
Quando você cola uma API Key no terminal, o cursor pode não se mover e nada aparece na tela. Isso é normal -- o terminal esconde a chave por segurança. Pressione Enter mesmo assim.

O que o assistente vai perguntar

  1. ANTHROPIC_API_KEY: Cole sua chave da Anthropic (começa com sk-ant-...) e pressione Enter
  2. OPENAI_API_KEY: Cole sua chave da OpenAI (começa com sk-...) e pressione Enter
  3. Para qualquer outra pergunta opcional: pressione Enter sem digitar nada (aceita o valor padrão)

Como verificar

O assistente deve terminar sem mensagens de erro em vermelho. Você vai ver algo como "Setup complete!" ou "Configuração concluída".

Arquivos de configuração (.env) foram criados na pasta mega-brain.

Se deu certo

$ npx mega-brain-ai setup Setting up Mega Brain... ANTHROPIC_API_KEY: ******** OPENAI_API_KEY: ******** Setup complete! Configuration saved to .env

Se aparecer isso

$ npx mega-brain-ai setup ERR_INVALID_API_KEY: The API key provided is invalid -- ou -- Cannot find module 'mega-brain-ai'

npm: command not found

O Node.js não foi instalado corretamente. Volte para a Etapa 3 e reinstale.

ENOENT: no such file or directory package.json

Você não está dentro da pasta mega-brain. No terminal, verifique onde está com pwd (Mac) ou cd (Windows) e navegue até a pasta correta.

ERR_INVALID_API_KEY durante o setup

A chave que você colou está errada ou incompleta. Abra o console.anthropic.com ou platform.openai.com, crie uma nova chave e tente novamente.

npx mega-brain-ai: command not found

Tente rodar npm install -g mega-brain-ai primeiro, depois tente o setup novamente.

~4 minutos

Etapa 8: Primeiro Uso -- Validação

Na etapa anterior, você configurou o Mega Brain com suas API Keys.
Agora vamos confirmar que tudo funcionou. Um único comando vai verificar o sistema inteiro e te dizer se está tudo certo.
Antes de começar esta etapa:
  • Etapa 7: Mega Brain configurado

Este é o momento de verdade. O comando /jarvis-briefing é como um exame médico para o Mega Brain -- ele verifica cada parte do sistema e te dá um score de 0 a 100.

Abra o Claude Code dentro do VS Code. Ele deve estar no painel lateral ou pode ser aberto pelo menu de extensões.

No campo de chat do Claude Code, digite:

/jarvis-briefing

Aguarde -- o sistema vai analisar sua configuração e retornar um relatório.

O que o score significa

Score 70 a 100 (verde): Tudo funcionando perfeitamente. Você está pronto para usar o Mega Brain.

Score 40 a 69 (amarelo): A maioria funciona, mas algo pode estar faltando. O briefing vai mostrar o que precisa de atenção.

Score abaixo de 40 (vermelho): Algo importante não está configurado. Leia o relatório -- ele explica exatamente o que falta.

/jarvis-briefing não funciona / comando não reconhecido

Certifique-se de que você está digitando no chat do Claude Code (a extensão), não no terminal do VS Code. São janelas diferentes.

Score abaixo de 40

Leia o relatório completo do briefing -- ele lista especificamente o que está faltando. Os problemas mais comuns são API Keys incorretas (volte para Etapa 6) ou npm install não completou (volte para Etapa 7).

O Claude Code não responde

Verifique se sua conta Claude está ativa e com crédito disponível em claude.ai.

Etapa 9: GitHub (Opcional)

Bonus
Você já pode usar o Mega Brain! Esta etapa é um bônus para quem quer salvar seu trabalho na nuvem. Se preferir, pode fazer isso em outro momento.
Na etapa anterior, você validou que o Mega Brain está funcionando com o /jarvis-briefing.
Esta etapa e opcional -- o Mega Brain funciona perfeitamente sem ela. Se quiser salvar seu trabalho no GitHub, siga os passos abaixo.

Criar sua Conta GitHub

O GitHub é onde os projetos de código ficam guardados na nuvem. Com uma conta, você nunca perde o trabalho que fizer no Mega Brain.

Acesse o site do GitHub para criar sua conta:

Acessar github.com
  1. Acesse github.com e clique em Sign up
  2. Digite seu email e crie uma senha
  3. Escolha um username (nome de usuario) -- esse nome vai aparecer nos seus projetos
  4. Confirme seu email quando chegar a mensagem de confirmação

Configurar sua Identidade no Git

Com o terminal do VS Code aberto, configure seu nome e email para o Git:

git config --global user.name "Seu Nome"

Substitua Seu Nome pelo seu nome real.

git config --global user.email "seu@email.com"

Use o mesmo email da sua conta GitHub.

Como verificar

Execute o comando abaixo para conferir que tudo ficou configurado:

git config --list

Você deve ver seu nome (user.name) e email (user.email) na lista.

Não recebi o email de confirmação do GitHub

Verifique a pasta de spam. Se não chegou em 10 minutos, tente se cadastrar novamente.

git config: command not found

O Git não está instalado. Volte para a Etapa 5 e instale o Git primeiro.
